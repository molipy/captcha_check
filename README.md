# captcha_check
基于深度学习的【验证码识别系统】设计与实现~Python

## 一、介绍
验证码识别，使用Python作为开发语言，通过TensorFlow搭建CNN卷积神经网络算法模型，并通过对收集的几千张验证码图片作为数据集，然后进行迭代训练，最终得到一个识别精度较高的模型文件，然后使用Django框架搭建网页端操作界面，实现用户上传一张验证码图片识别其名称。
